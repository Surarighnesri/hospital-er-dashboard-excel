# 🏥 Hospital Emergency Room Dashboard (Excel)

## 📊 Overview
This project is an Excel dashboard built to analyze emergency room patient data.

## 🔍 Key Metrics
- Total Patients: 464
- Average Wait Time: 34.72 mins
- Patient Satisfaction Score: 4.99
- Admission Rate: 51%

## 📈 Features
- Monthly filtering using slicers
- Age group analysis
- Department-wise referrals
- Gender distribution
- Patient attendance tracking

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualizations
- Data Cleaning
